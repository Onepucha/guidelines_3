Изменение навигации курса

Для изменения навигации курса необходимо перейти на мастер-слайд и изменить текстовый элемент menu. По дефолту все пункты меню заблокированы. Чтобы сделать пункт активным, нужно убрать class disable. Чтобы задать переход на нужный кадр, необходимо в событии onclick вызвать метод gotoFrame('ID_FRAME'), где ID_FRAME — это идентификатор кадра. Для корректной работы меню необходимо, чтобы количество слайдов соответствовало количеству пунктов меню.
Пример первого пункта меню:

Первый пункт меню 

                    
<p class="contents_item-list title first js-nav-link" data-frame="FRAME_37">
    Первый пункт меню
</p>
Пример заблокированного пункта меню:

Заблокированный пункт меню

            
<div class="contents_item-list js-nav-link disabled" data-frame="ID_FRAME">
    <p>Заблокированный пункт меню</p>
</div>
