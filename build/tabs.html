Вкладки / табы (.tabs)

Набор вкладок содержит несколько страниц, на которые можно поместить другие элементы, такие как текстовые поля или картинки. Когда пользователь нажимает соответствующую вкладку, эта страница становится активной.
  • Вертикальные вкладки слева
  • Вертикальные вкладки справа
  • Горизонтальные вкладки сверху
  • Горизонтальные вкладки снизу
  • Структура
  • Предварительный выбор вкладки
  • Инициализация (запуск) вкладок

Структура

В блоке с классом tab__nav содержатся вкладки, каждая из которых имеет параметр data-href, соответствующий ее идентификатору (id).
При нажатии на вкладку виден только контейнер с соответствующим идентификатором.
Каждая вкладка должна ссылаться на свой блок внутри tab__content, поэтому количество вкладок и блоков должно совпадать.
              
<div class="tab__nav ">
    <a class="tab__links" data-href="#tab-content-1">Имя вкладки</a>
</div>
<div class="tab__content">
    <div id="tab-content-1" class="tab__item">
        Содержимое страницы
    </div>
</div>

Предварительный выбор вкладки

По умолчанию выбрана первая вкладка. Но вы можете заранее выбрать вкладку, добавив класс active в тэг a.
              
<a class="tab__links active" data-href="#tab-content-2">Имя вкладки</a>

Инициализация (запуск) вкладок

Для этого нужно в действиях объекта TXT_№ на вкладке «После показа» выбрать javascript и внутри выполнить инициализацию и поверку на разблокировку навигации.
              
tabs();

$("body").on("click", ".tab__links", function () {
    if ($('.tab__links:visible.visited').length == $('.tab__links:visible')
        .length) {
        $('.btn__primary.nav.disabled:visible').parents().eq(3).hide();
    }
});
