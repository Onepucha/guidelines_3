Плитка с текстом

Плитка с обводкой для предоставления информации. Для отображения элемента создайте блок с классом plate, в который помещается информация.
Пример:

1

Ситуационные

Общие вопросы о текущей ситуации.
            
<div class="plate plate--icon" style="width: 275px;">
    <div class="divider left">
        <div class="divider--text">
            <h1 class="h1 text-light-gray">1</h1>
        </div>
    </div>
    <h4 class="h4">Ситуационные</h4>
    <div class="text text-xl">Общие вопросы о текущей ситуации.</div>
</div>

Плитка с иконкой

Для правильного отображения иконки внутри плитки нужно к блоку с классом plate добавить класс plate—icon (он отвечает за иконку внутри блока). Также внутрь блока с классом plate plate—icon нужно добавить блок с классом divider. У этого класса есть 2 преднастройки позиционирования картинки:
  • класс left позиционирует картинку по левому краю;
  • класс right — по правому.
Внутри блока с классом divider left или divider right должен быть блок с классом divider—text, в который помещается иконка или нужный текст.
Пример:

Департамент брокерского обслуживания банка Компания

Брокерские услуги для розничных клиентов.
            
<div class="plate plate--icon" style="width: 275px;">
    <div class="divider center">
        <div class="divider--text">
            <img src="./images/icon-1.png" alt="icon-1">
        </div>
    </div>
    <h4 class="h4">Департамент брокерского обслуживания банка Компания</h4>
    <div class="text text-xl">Брокерские услуги для розничных клиентов.</div>
</div>

Плитка с разделителем

Чтобы добавить разделители между картинкой и текстом, нужно вставить внутрь блока с классом divider первым элементом тэг span с классом divider-border before (т.е. перед), после чего разместить блок с классом divider—text с картинкой внутри блока, и добавить заключительный тэг span с классом divider-border after (т.е. после).
Пример:
Плитка с разделителем.
                        
            <div class="plate plate--icon" style="width: 275px;">
                <div class="divider">
                    <span class="divider-border after"></span>
                    <div class="divider--text">
                        <img src="./images/icon-1.png" alt="icon-1">
                    </div>
                    <span class="divider-border before"></span>
                </div>
                <div class="text text-xl">Плитка с разделителем.</div>
            </div>

Кликабельные плитки

Чтобы вызвать модальное окно, нужно навесить событие onclick на блок plate, где #TXT_* — идентификатор блока onclick="$('#TXT_*').show()", пример можно посмотреть в разделе "Модальные окна (.modal)"
Пример:
Плитка с нажатием
                        
<div class="plate plate--icon modal-open" style="width: 275px;" onclick="$('#TXT_ID_БЛОКА').show()">
    <div class="divider">
        <span class="divider-border after"></span>
        <div class="divider--text">
            <img src="./images/icon-1.png" alt="icon-1">
        </div>
        <span class="divider-border before"></span>
    </div>
    <div class="text text-xl">Плитка с нажатием</div>
</div>
