Переворачивающиеся карточки (.features)

На лицевой стороне изображена иконка и заголовок того, о чем идет речь в тексте на обратной стороне. Данный элемент выделяется круговыми стрелками в правом нижнем углу.
            
<div class="features">
    <a href="" class="features--item">
        <div class="features--item__inner">
            <div class="features--front">
                <div>
                    <img class="fluid mb-pink" src="./images/people.png" alt="">
                    <h4 class="h4 text-blue text-center">Люди</h4>
                </div>
            </div>
            <div class="features--back">
                <div>
                    <div class="text text-s">Активный подход к управлению портфелями, обеспечивающий генерацию
                        добавочной доходности без претензий на вмешательство в корпоративное управление.</div>
                    <div class="text text-s">Основа инвестиционного подхода – фундаментальный анализ.</div>
                </div>
            </div>
        </div>
    </a>
</div>

Люди

Активный подход к управлению портфелями, обеспечивающий генерацию добавочной доходности без претензий на вмешательство в корпоративное управление.
Основа инвестиционного подхода — фундаментальный анализ.
Чтобы на экране отобразились элементы, нужно создать блок div с классом features, внутри которого будут находиться элементы карточек.

features—item

Отвечает за сам элемент карточки шириной в 1/3.

features--item__inner

В нем расположены два блока с классами features--front и features—-back.

features с префиксом –-front

Отвечает за лицевую сторону.

features с префиксом -–back

Отвечает за обратную сторону с наполнением.
