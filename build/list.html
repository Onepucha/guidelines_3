Списки (.list__ul / .list__ol)

  • Маркированный список
  • Нумерованный список

Маркированный список

Для маркированного списка используется тэг ul с классом list__ul и класс, отвечающий за размер текста в списке (например, text-m). Каждый элемент маркированного списка должен иметь класс list__ul-item,, отвечающий за стилизацию списка.
Пример:
  • Пункт 1.
  • Пункт 2.
            
<ul class="list__ul text text-m">
    <li class="list__ul-item">Пункт 1.</li>
    <li class="list__ul-item">Пункт 2.</li>
</ul>
Для списка со знаками «+» и «-» нужно поменять префикс, если нужен «+», то префикс -plus, если нужен минус, то префикс -minus.
Пример списка с «+»:
  • Пункт 1.
  • Пункт 2.
            
<ul class="list__ul text text-m">
    <li class="list__ul-plus">Пункт 1.</li>
    <li class="list__ul-plus">Пункт 2.</li>
</ul>
Пример списка с «-»:
  • Пункт 1.
  • Пункт 2.
            
<ul class="list__ul text text-m">
    <li class="list__ul-minus">Пункт 1.</li>
    <li class="list__ul-minus">Пункт 2.</li>
</ul>

Нумерованный список

Для нумерованного списка используется тэг ol с классом list__ol и класс, отвечающий за размер текста в списке (например, text-m). Каждый элемент нумерованного списка должен иметь класс list__ol-item, отвечающий за стилизацию списка.
Пример:
  1. Пункт 1.
  2. Пункт 2.
            
<ol class="list__ol text text-m">
    <li class="list__ol-item">Пункт 1.</li>
    <li class="list__ol-item">Пункт 2.</li>
</ol>
