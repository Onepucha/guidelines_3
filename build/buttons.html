Для скачивания документа

При нажатии на эту кнопку скачивается какой-либо документ, чаще всего в формате PDF. 
                
<button class="btn btn__primary dark-blue btn__round m"
    onclick="openRequestedPopup('./images/pdf/Fond.pdf')">
    Скачать
</button>
Чтобы скачать файл по клику на кнопку, нужно подготовить файл, который именуется латинскими буквами без пробелов.
По клику на кнопку вызывается функция openRequestedPopup, в которую аргументом функции мы передаем строку, содержащую ссылку на документ или ресурс.

Для перехода между слайдами

На каждом слайде в нижней правой части есть счетчик слайдов и кнопки перехода между ними. Если на слайде есть разговорная выноска / текстовый баллон .speech, то кнопка «Продолжить» становится некликабельной, чтобы на слайде не было 2 кнопки с одинаковыми функциями.
25 / 159
В этих кнопках можно менять цвет стрелки. Для этого в тегах icon__left (стрелка слева) и icon__right (стрелка справа) добавьте название цвета.
Стандартным считается white, но можно заменить его на:
  • red
  • blue
  • dark-blue
  • gray
                        
<button class="btn__prev">
    <svg class="icon__left red" width="10" height="18" viewBox="0 0 10 18" fill="none"
        xmlns="http://www.w3.org/2000/svg">
        <path d="M9 1L1 9L9 17" stroke="white" stroke-width="1.5" stroke-linecap="round" stroke-linejoin="round" />
    </svg>
    <span>Назад</span>
</button>
<div class="slide__counter">
    <span>25</span>
    <span>/</span>
    <span>159</span>
</div>
<button class="btn__primary nav"><span>Продолжить</span>
    <svg class="icon__right" xmlns="http://www.w3.org/2000/svg" width="10" height="18" viewBox="0 0 10 18" fill="none">
        <path d="M1 1L9 9L1 17" stroke="white" stroke-width="1.5" stroke-linecap="round" stroke-linejoin="round" />
    </svg>
</button>
<button class="btn__primary nav disabled"><span>Продолжить</span>
    <svg class="icon__right gray" xmlns="http://www.w3.org/2000/svg" width="10" height="18" viewBox="0 0 10 18"
        fill="none">
        <path d="M1 1L9 9L1 17" stroke="white" stroke-width="1.5" stroke-linecap="round" stroke-linejoin="round" />
    </svg>
</button>

Подсказка

Работает так же, как и кнопка для вызова модального (всплывающего) окна. Разница лишь во внешнем виде: подсказка не содержит текста и занимает меньше места.
