Описание

Для правильного отображения карусели нужно в блок с классом single-item добавить нужное количество блоков с классом slider_item, которые наполняются необходимым контентом.

Инициализация

Чтобы инициализировать (запустить) слайдер, нужно на объекте TXT_№ в действиях объекта на вкладке «После показа» выбрать действие javascript и внутри выполнить инициализацию.
Инициализация выглядит следующем образом:
            
slider();
