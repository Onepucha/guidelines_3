Далее
Структура выноски состоит из главного блока с классом speech и дополнительных классов:
down — устанавливает направление выноски снизу.
up — устанавливает направление выноски сверху.
Классы down и up не могут существовать без классов, которые устанавливают положение выноски:
right__top — справа сверху.
right__midldle — справа по центру.
right__bottom — справа снизу.
left__top — слева сверху.
left__middle — слева по центру.
left__bottom — слева снизу.
Цвет выноски:
По умолчанию цвет заливки серый.
white — устанавливает белый цвет заливки.
Закругление блока:
radius — устанавливает радиус скругления.
Класс manySpeech предназначен для выноски со сменой текста внутри него, то есть все блоки будут показаны поочередно по клику на кнопку «Далее». Кнопка обязательно должна содержать класс btn__block и js-nextSpeech.
