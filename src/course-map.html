Ветвление / карта курса

В курсе можно сделать нелинейную структуру. Для этого есть специальные кликабельные плитки, при нажатии на которые мы попадаем в другую тему, состоящую из одного или нескольких слайдов.
После изучения темы мы попадаем обратно на слайд с ветвлением, и кнопка пройденного блока окрашивается в другой цвет.
После изучения всех тем кнопка «Продолжить» для перехода на следующий слайд разблокируется.

Описание

Для перехода на нужный фрейм заполните дата атрибут data-frame="Фрейм_для_перехода". Чтобы сделать плитку активной, нужно удалить класс visited в строке plate plate--icon plate--nav visited.

Инициализация

Чтобы инициализировать карту перемещений, нужно навесить событие на заблокированную кнопку с идентификатором  TXT_№. Для этого в действиях объекта на вкладке «После показа» выберите действие javascript и внутри выполните инициализацию.
