Раскрывающийся список (.accordion)

Раскрывающийся список — это элемент, который раскрывается при нажатии. Позволяет скрывать часть информации на слайде до тех пор, пока она не пригодится.

</div>

Структура

Структура раскрывающегося списка состоит из неупорядоченного списка вкладок accordion__title, который соответствует количеству блоков с классом accordion__content.
У каждого блока с классом accordion рядом должен быть блок с классом accordion__title и accordion__content. Если нажать на вкладку, становится видимым только тот контейнер, к которому она относится.

Инициализация

Чтобы инициализировать (запустить) раскрывающийся список, нужно на объекте TXT_№ в действиях объекта на вкладке «После показа» выбрать действие javascript и внутри выполнить инициализацию.
Инициализация выглядит следующем образом:
            
$(".accordion").accordion({ collapsible: true, heightStyle: "content", active: false});
