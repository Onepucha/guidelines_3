Модальные окна

Для вызова всплывающего (модального) окна

При нажатии на такую кнопку по центру экрана возникает модальное окно, содержащее какую-либо информацию. Оставшаяся часть вне этого окна затемняется. 
            
<button class="btn btn__primary btn__round m btn__modal modal-close">Модальное окно с текстом</button>
Используйте модальное окно для диалоговых окон или другого содержимого, которое можно вызвать.
Чтобы модальное окно работало, нужно вызвать событие — показ определенного TXT_№. Чтобы создать кнопку закрытия, добавьте класс modal-close к своей кнопке.
Чтобы модальное окно отображалось правильно, нужно в блок с классом modal добавить блок с классом modal-content (если это модальное окно с текстом), в котором находится все наполнение. Блок с классом btn__modal modal-close должен находится вне блока с классом modal-content.
Чтобы вызвать модальное окно, также можно использовать событие onclick, где #TXT_* идентификатор блока onclick="$('#TXT_344').show()"
Пример кода модального окна с кнопкой снизу:
                        
<div class="modal">
    <div class="modal-content">
        <div class="h5 mb-pink">Пример заголовка в модальном окне с текстом.</div>
        <div class="text text-l">Пример текста в модальном окне с текстом.</div>
    </div>
    <div class="btn btn__primary btn__round m btn__modal modal-close">Назад
    </div>
</div>
